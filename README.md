### 🛡️ [SafeGuard AI — Industrial PPE & Tool Safety System](https://github.com/kbvinay001/Safeguard-AI)
> **Problem:** Thousands of industrial accidents occur annually due to PPE non-compliance and abandoned tools — existing monitoring is manual and slow.

Three simultaneous **YOLOv11n** models running on GPU, fused spatially via IoU matching to associate PPE items to specific workers — not just the scene.

| Model | Task | mAP@50 |
|---|---|---|
| HUMAN | Worker / person detection | **99.44%** |
| PPE | Helmet · Vest · Gloves · Goggles | **79.90%** |
| TOOLS | Drill · Hammer · Pliers · Wrench abandonment | **67.87%** |

- **Temporal FSM** — T1=25s WARNING → T2=35s CRITICAL; reduces false alarms by ~34% vs instant-alert systems
- **Dynamic Hazard Zones** — each tool's hazard zone scales with its bounding box (1.8× expansion factor)
- **Vision-only** — works on any existing CCTV, no IoT sensors, RFID, or wearables needed
- Streamlit cyberpunk HUD dashboard + FastAPI REST backend + SQLite + PDF safety report generator
- **Tech:** `Python` `YOLOv11` `Ultralytics` `OpenCV` `FastAPI` `Streamlit` `SQLite` `PyTorch` `CUDA FP16`

### 🚦 [Green Wave++ — Emergency Vehicle Traffic Preemption](https://github.com/kbvinay001/Green-wave)
> **Problem:** Ambulances lose critical minutes at red lights — no real-time AI system pre-clears corridors before the vehicle even arrives.

Certainty-aware multi-modal system detecting approaching ambulances from **CCTV and microphone arrays simultaneously**, fusing evidence with a temporal belief engine to cascade green lights before the vehicle reaches the intersection.

```
Microphone array → CRNN siren detector → GCC-PHAT bearing ─┐
                                                             ├→ TemporalFusionEngine → SUMO Green Wave
CCTV camera     → YOLOv11 detector → lane assigner ─────────┘
```

- **GCC-PHAT TDOA** multi-mic array bearing estimation for audio source localisation
- **Temporal belief fusion** — Gaussian-weighted audio + vision, exponential decay, hysteresis prevents false triggers
- **Certainty gating** — belief must exceed threshold and hold for `arm_duration_sec` before preemption fires
- React + Vite live dashboard over WebSocket with bearing compass, intersection map, and event feed
- **Tech:** `Python` `PyTorch` `YOLOv11` `CRNN` `GCC-PHAT` `RAFT` `FastAPI` `React` `Vite` `SUMO` `TraCI`
